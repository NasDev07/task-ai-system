================================================================================
RECORDING TIPS:
================================================================================

AUDIO:
- Speak clearly and naturally
- Use a quiet environment
- Moderate pace - not too fast, not too slow
- Let silence exist during AI processing (shows it's working)

VIDEO:
- Record at 1080p or higher
- Move mouse slowly and deliberately
- Highlight important buttons and sections
- Show error handling if something fails

TIMING:
- Aim for 9-10 minutes total
- Allow 2-3 seconds for actions to complete
- Don't rush the AI generation - let viewers see it happening

EDITING:
- Keep cuts minimal and natural
- Add captions for key features (optional)
- Include intro and outro with background music (optional)
- Remove long pauses between actions

UPLOAD:
- Save as MP4 or similar format
- Upload to YouTube
- Make Public or Unlisted
- Add description with:
  * Live URL
  * GitHub link
  * Demo credentials
  * Brief summary
